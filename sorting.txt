##BUBBLE SORT##
Given an array of n elements:
1.Start from the first element.
2.Compare adjacent elements.
3.If the left element is greater than the right element, swap them.
4.Continue until the end of the array.
5.Repeat the process for n-1 passes or until no swaps occur.

##INSERTION SORT##
1.Assume the first element is already sorted.
2.Pick the next element (called the key).
3.Compare the key with elements to its left.
4.Shift larger elements one position to the right.
5.Insert the key into its correct position.
6.Repeat until all elements are sorted.

##SELECTION SORT##
1.Find the smallest element in the array.
2.Swap it with the first element.
3.Find the smallest element in the remaining unsorted array.
4.Swap it with the second element.
5.Repeat until the array is sorted.

##DUTCH FLAG ALGORITHM##
1. Initialize three pointers low, mid and high to 0, 0 and n-1 respectively.
2. Traverse the array from start to end using the mid pointer.
3. If the element at mid is 0, swap it with the element at low and increment both low and mid pointers.
4. If the element at mid is 1, just move the mid pointer to the next element.
5. If the element at mid is 2, swap it with the element at high and decrement the high pointer.
6. Repeat steps 3-5 until mid pointer is less than or equal to high pointer.

##Custom sort##
A Custom Sort means sorting elements according to a user-defined rule instead of the default ascending or descending order.
In programming, this is usually done by providing a custom comparator function.

Steps
1.Define a comparison rule (comparator).
2.For every pair of elements a and b:
3.Return negative/true if a should come before b.
4.Return positive/false if b should come before a.
5.Apply any sorting algorithm (TimSort, Merge Sort, Quick Sort, etc.) using this comparator.
6.The array is arranged according to the custom rule.
